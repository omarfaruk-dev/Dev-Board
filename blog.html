Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where you attach a single event listener to a parent element, rather than attaching individual event listeners to each of its child elements. This allows the parent to handle events that occur on its descendants.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the Document Object Model (DOM) where an event that is triggered on an element will "bubble up" through its ancestors in the DOM hierarchy. This means the event will first be handled by the innermost element (the target), and then it will propagate up to its parent elements, all the way to the root of the document (usually the document object itself), unless it's stopped.
